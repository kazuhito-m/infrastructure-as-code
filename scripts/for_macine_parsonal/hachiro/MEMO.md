# hachiro サーバのお試しいろいろ。

## Growiデータをインポートする

少し癖があるので、手順を記載

1. 設定のアプリ設定から、メンテナンスモードに設定する
2. 設定のデータインポートに、バックアップファイルを読み込ませる
3. 全てにチェックを付け、出来る限り `Flush and insert` に変える
    - ただし、`Pages` と `Revisions` については `Upsert` に設定する
4. インポートを実行する
